Summary

PAX explores how a cache-coherent accelerator can provide crash consistency for persistent-memory data structures. The accelerator interposes on persistent memory operations, performs undo logging, and preserves the direct-access benefits of persistent memory without requiring software to execute logging code on the critical path.
