Summary

This article revisits RAMCloud's networking and request-dispatch design in the context of low-latency RDMA-based storage systems. It explains why RAMCloud uses conventional RPC-style server processing rather than one-sided RDMA, and uses Rocksteady to show how richer storage functionality can still exploit modern NIC features.
